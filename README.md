# liri-node-app

LIRI is a _Language_ Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data

LIRI can take in one of the following commands:

   * `concert-this`

   * `spotify-this-song`

   * `movie-this`
